# ROvACC Discord Bot

## Overview

This is the code that generates the ROvACC Discord Bot. It is build on the NestJs framework


## Installation

Clone the repository

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
